# 簡易仕様書

### 作者
ルンノー
### アプリ名
Flashcard Maker

#### コンセプト
勉強用単語帳アプリ

#### こだわったポイント
ランダムでカードの表示。
また、カードを押すとカードのうらを表示するようにしました。
SQLiteでのデータ処理。

### 該当プロジェクトのリポジトリ URL（GitHub,GitLab など Git ホスティングサービスを利用されている場合）
https://github.com/lumnaw11/Flashcard-Maker-Android-App-SQLite

## 開発環境
### 開発環境
Android Studio (Electric Eel)

### 開発言語
Java

## 動作対象端末・OS
### 動作対象OS
Android SDK 32

## アプリケーション機能

### 機能一覧
- カードの表示
- ランダムモード
- カードの追加、更新、作成、削除、または全部のカードの削除。

### 画面一覧
- スタート画面 ：カード画面に移動とカードを追加する画面に移動する
- カード画面 ：単語をカード式に表示する。ランダム表示モード
- エディットカード画面　：カードの追加、更新、作成、削除、または全部のカードの削除。

### 使用しているAPI,SDK,ライブラリなど
- データの処理はSQLiteでやりました。
